Descripción:
La función buscar_contenido_por_nombre_o_autor es una función en PostgreSQL que permite buscar en la tabla CONTENIDO aquellos registros cuyo nombre o autor coincidan con un patrón de búsqueda. Utiliza expresiones regulares para hacer coincidencias flexibles y case-insensitive tanto en el campo nombre como en el campo autor.

Input:
pattern (TEXT):
Un patrón de búsqueda en formato de expresión regular. Este patrón se compara con los campos nombre y autor de la tabla CONTENIDO utilizando una coincidencia case-insensitive.


Output:
id (INT):
Retorna el id del registro que coincide con el patrón de búsqueda.

nombre (VARCHAR):
Retorna el nombre del registro que coincide con el patrón de búsqueda.

Ejemplo de Uso:
SELECT * FROM buscar_contenido_por_nombre_o_autor('medicina deportiva');
