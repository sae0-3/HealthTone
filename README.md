# HealthTone

### Prerequisitos

* Tener instalado `docker`

### Hacer Correr
```hs
git clone git@github.com:sae0-3/healthtone
```

```hs
cd healthtone
```

```hs
docker compose up -d --build
```

Para finalizar
```hs
docker compose down
```
